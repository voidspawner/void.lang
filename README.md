# V O I D lang

## About
**‚åú V O I D lang ‚åü** is the language for rapidly creating applications and games in the **[V&nbsp;O&nbsp;I&nbsp;D](#v-o-i-d-format)** or **[JSON](https://en.wikipedia.org/wiki/JSON)** format. It is used as a replacement for the standard **``Bash``**„Éª**``CMD``**„Éªetc. languages and for creating **``games``**, **``apps``** and **``content``**. The language uses one of the languages already preinstalled in the system. So you don't need to install anything else. Code and data are not separated. The whole application fits in **one ``V O I D`` or ``JSON`` file**. Since the **code is presented as data**, applications can be easily generated with **``AI``**, updated, installed and launched remotely.

## How to Use

1. Download **V&nbsp;O&nbsp;I&nbsp;D lang**
2. Create your first app in **``app.void``**„Éª**``app.json``**„Éª**``app.zip``**
3. Launch app with **V&nbsp;O&nbsp;I&nbsp;D lang**
 
```console
python3 void.py app.void
python3 void.py app.json
python3 void.py app.zip
```

> [!TIP]
> Create an app in **``run.void``**„Éª**``run.json``** „Éª**``run.zip``** for autorun
> ```
> python3 void.py
> ```

> [!TIP]
> Type alias in the **``Windows``** command line or PowerShell
> ```console
> doskey void=python3 c:\path\to\void.py
> ```
> ```console
> function void { python3 c:\path\to\void.py @args }
> ```
>
> Add alias in **``~/.bashrc``** (Linux)„Éª**``~/.bash_profile``** (macOS)„Éª**``~/.zshrc``** (macOS)
> ```console
> alias void="python3 /path/to/void.py"
> ```
> ```console
> void app.void
> ```

> [!TIP]
> To install Python on **``Windows``**, press **``Win``** + **``R``** and paste the command
> ```console
> winget install --id Python.Python -h --accept-package-agreements --accept-source-agreements
> ```
> To install Python on **``macOS``** or **``Windows``**, enter the command in the terminal and follow the instructions
> ```console
> python3
> ```

> [!TIP]
> Compile **``Python``**„Éª**``JavaScript``**„Éª**``Swift``**„Éª**``Kotlin``**„Éª**``C++``** file containing your code into the exacutable
> ```console
> pip3 install pyinstaller
> pyinstaller --onefile void.py
> ```
> ```console
> npm install -g pkg
> pkg void.js
> ```
> ```console
> swiftc void.swift
> ```
> ```console
> kotlinc void.kt
> ```
> ```console
> clang++ void.cpp -o void.exe
> ```

> [!TIP]
> Convert your **app**„Éª**game** to various platforms
> ```console
> void.exe convert app.void app.exe
> void.exe convert game.void game.exe
> ```

> [!TIP]
> Import **``void.py``**„Éª**``void.js``**„Éª**``void.swift``**„Éª**``void.kt``** into the project to create an app
>
> Import **``void.gd``**„Éª**``void.cpp``** into the **Godot**„Éª**Unreal Engine** project to create a game

## V O I D format
This is a data format that inherits the best features of [**JSON**](https://en.wikipedia.org/wiki/JSON), [**YAML**](https://en.wikipedia.org/wiki/YAML), [**CSV**](https://en.wikipedia.org/wiki/Comma-separated_values) and [**plain text**](https://en.wikipedia.org/wiki/Plain_text) formats. Makes it easier to write and read data, both by human and by program. The format simplifies data creation by removing the use of unnecessary quotation marks, brackets, colons, commas and other symbols. It is possible to combine **text** and **binary** data.
